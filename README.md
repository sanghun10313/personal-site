# SANGHUN Personal Site

Vite + React + Tailwind 기반의 퍼스널 웹사이트.

## 로컬 실행
```bash
npm i
npm run dev
```
브라우저에서 http://localhost:5173 열기

## 빌드
```bash
npm run build
npm run preview
```

## Vercel 배포
1. Vercel 로그인 (GitHub 연동)
2. **Add New Project** → GitHub 저장소 선택
3. Framework: **Vite** (자동 인식됨), Build Command: `npm run build`, Output: `dist`
4. **Deploy** 클릭
